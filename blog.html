Study Choice

I chose HBO-ICT because I have always been interested in how computers and the programs on them work. I like to find out how the different components work together and to play around with this. I have also tried to make small games with the help of Youtube tutorials.

At the moment it is not entirely clear to me what I am going to do after my studies, but it will most likely be something within software development or BIC.

As for how I started at HZ. I have visited several schools including HZ(HBO-ICT) and Avans(Informatica / Technical Informatics) and made some online tests. At first I chose Avans(Informatica), but after a year I decided to start at HZ(HBO-ICT).

Programming experience

Before I started at HZ I studied for a year at Avans HBO Informatica. There, I made several projects. These projects involved:

  • A website (made with HTML, CSS, JS and Bootstrap).
  • A Windows application written in Java with a MySQL database link.
  • An Android application written in Java where information about movies and series was retrieved and displayed using an API. (TheMovieDB).
  • An API written in JS, This used Node.js, Npm, Mocha, Chai, MySQL and JWT.
  • A PWA written in Vue.js, PHP, JS and HTML (This was a project for an employer. This project is online and can be found at https://www.1t3.nl/).

    • For many of these projects, we were taught how to work with Git. Besides these project I tried to make a small game in Unity using YouTube tutorials.

    For our first feedback moment, we had to create a website that reflected various information about the course. We then had to present this website to a group of students and teachers. The feedback I received on this was mainly from the teachers present. The feedback I received concerned:

  • The lack of a blog page
  • The lack of a progress bar
  • The use of capitals in file names
    • I then applied this feedback to improve my site.
